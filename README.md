# Study Hall

A multi-subject, multi-student AI-powered study app.

## Setup

### 1. Supabase
Run `SUPABASE_SETUP.sql` in your Supabase SQL editor.

### 2. Vercel Environment Variables
In Vercel → Project Settings → Environment Variables, add:

| Name | Value |
|------|-------|
| `VITE_SUPABASE_URL` | Your Supabase project URL |
| `VITE_SUPABASE_KEY` | Your Supabase publishable key |
| `ANTHROPIC_API_KEY` | Your Anthropic API key |

### 3. Deploy
Connect this GitHub repo to Vercel. It will build and deploy automatically.
